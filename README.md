# Uranium Age
  A minecraft mod that adds _Uranium_.

## Description
  This mod adds _Uranium_ and reactors to the game, which the player can use and benifit from. For a more detailed description check out my [Modrinth page](https://modrinth.com/mod/uranium-age).

## How to use?
  Install the mod's .jar file from modrinth and paste/add it into your minecraft instance's Mod folder. This mod currently only works on the the minecraft version 26.1.2 using the fabric loader and fabric api, so it will only work on a fabric minecraft isntance.
